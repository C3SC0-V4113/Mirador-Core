# Diagramas

Los Mermaid en `docs/diagrams/mermaid` son la fuente textual inicial para revisar
arquitectura, flujo MCP y modelo de datos.

## Archivos

- `architecture.mmd`: componentes principales del sistema.
- `mcp-flow.mmd`: separacion entre MCP adapter y backend core.
- `database-model.mmd`: entidades esperadas de datos y conocimiento.

Los diagramas draw.io editables se agregaran despues de estabilizar la arquitectura
de implementacion.
